World's 10 Richest People in 2024

This project's goal is to create a Javascript Sortable List App

  • Create an ordered list (Top 10 richest people)
  • Scramble list items randomly
  • Allow user to drag and drop an item to a different position
  • Button to check if items are in the correct order
  • Show green for Correct order and red for Wrong order

What I practiced in this project:

  • DOM manipulation
  • Functions in Javascript
  • Event listeners
  • Array Methods of Javascript
  • Adding icons
